# Pratinjau Formulir Invoice Publik

Untuk melihat tampilan formulir invoice publik yang sudah diperbarui:

1. Pastikan dependensi PHP sudah diunduh dengan `composer install`.
2. Jalankan server pengembangan Laravel dengan `php artisan serve`.
3. Akses `http://localhost:8000/invoices/public/create` melalui peramban.
4. Masukkan passphrase portal invoice jika diminta untuk mengakses formulir publik.

Tangkapan layar contoh pratinjau tersedia di dokumentasi perubahan terkini.
